# Internal project history

These documents preserve planning, investigation, and integrity-review history.
They describe multiple earlier repository states and are not current user
documentation or release claims.

The former enterprise roadmap and work breakdown are explicitly superseded by
[`ADR 0001`](../adr/0001-defer-scale-out-crawling.md), which keeps scale-out
architecture deferred pending measured need and benchmark evidence.

Current behavior is defined by the CLI help, the root README, automated tests,
and required CI gates. Open remediation work is tracked outside version control
during stabilization.
